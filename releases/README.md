## Release Types

- **Alpha** (`v0.x.x-alpha.x`) - Early development releases for testing
- **Beta** (`v0.x.x-beta.x`) - Feature-complete releases for community testing
- **Release Candidate** (`v0.x.x-rc.x`) - Final testing before stable release
- **Stable** (`v1.x.x`) - Production-ready releases
